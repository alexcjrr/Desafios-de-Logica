# Controle de Estoque e Carrinho de Compras

Este desafio simula duas funcionalidades comuns em sistemas de compras:

1. **Controle de Estoque**: Gerencia o estoque inicial de 5 unidades com possibilidade de adicionar ou retirar quantidades. Se a retirada for maior que o estoque disponível, uma mensagem de aviso é exibida.

2. **Carrinho de Compras**: Permite ao usuário informar quantos produtos deseja adicionar e insere os produtos em uma lista, exibindo o carrinho final.

### Regras

#### Controle de Estoque

- O usuário escolhe:
  - `1` para adicionar ao estoque.
  - `2` para retirar do estoque.
- O sistema atualiza o estoque ou exibe "Estoque insuficiente".

#### Carrinho de Compras

- O usuário informa o número de produtos.
- Em seguida, insere os nomes dos produtos.
- Ao final, o programa exibe os produtos inseridos.

### Exemplo de Saída

```
Estoque atualizado: 1
Carrinho final: pão, leite, açúcar
```
